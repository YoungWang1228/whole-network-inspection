# 电商全平台巡网打假系统
>按照指定关键词、指定商品图片的方式，自动从某宝、某东、并夕夕、某音、1688等知名电商平台搜索相关商品，
对比商品图片和商标图片找出疑似山寨、仿冒的产品和商家，交由人工后续维权打假。

通过系统自动化巡网和邮件预警的方式，品牌方可以节省人工巡网的人力成本，及时性也能得到保证。

![支持的电商平台](./images/1.png)

### 各平台支持的商品搜索方式
- 某宝：关键词、找相似、拍立淘
- 某东：关键词、拍立淘
- 并夕夕：关键词
- 某音：关键词
- 其他：大部分都只能用关键词搜索


### 巡网流程
1. 设置关键词，系统将使用关键词去电商平台搜索商品。某宝和某东还可以设置商品照片用拍立淘的功能找相似的商品
2. 设置用于比对的商品和logo、关键字，系统会把搜索商品结果和商家Logo通过图像比对的方式判断相似度，高度相似的标记为疑似。
   还会通过店铺名称、商品名称以及从商品主图和Logo图提取的文字进行关键字比对
3. 设置比对的商品数量，例如只比对前100个商品。可以节省时间和开销
4. 设置白名单，将有授权的商家或直营店加入白名单，系统会自动跳过
5. 选择需要巡网的电商平台和巡网频率，根据实际情况来选择，如3天一次或一周一次等
6. 巡网的结果可在结果库里查看，如果配置了预警接收邮箱，会收到预警邮件
